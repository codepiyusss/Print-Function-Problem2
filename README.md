# Print Function Challenge

A Python script that prints a sequence of consecutive integers from $1$ to $n$ as a single consolidated string, without utilizing any built-in string methods.

## Problem Description

The program reads an integer $n$ from standard input (`STDIN`) and outputs all integers from $1$ up to and including $n$ consecutively on a single line.

### Task
Without using any string methods, print the following sequence:
$$123\dots n$$

---

## Technical Specifications

### Constraints
* $1 \le n \le 150$

### Input Format
A single integer, $n$, read from `STDIN`.

### Output Format
Print the list of integers from $1$ through $n$ as a string, without any spaces between them.

### Examples
* **Input:** `3` $\rightarrow$ **Output:** `123`
* **Input:** `5` $\rightarrow$ **Output:** `12345`
